# iraira_de_amazon
学部1年の時にjavaで作成したイライラ棒ゲーム

案出し : 友人、自分
コーディング : 自分

# 遊び方
左側の青いタイルをクリックしてからスタートです。

カーソルを茶色の道から外したり、アニマル達に触れさせないようにしながら右端のお風呂をクリックするとクリアです。

進むごとに道は複雑になっていきます。
